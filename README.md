# appliance-service-web

Premium single-page style web app for booking **Refrigerator, AC, and Washing Machine** services.

## Features
- Login + signup pages with **Sign with Email** and **Sign with Google** buttons.
- Urban-company inspired premium UI with appliance image logos.
- Category-wise service page with pricing in **Indian Rupee (₹ INR)**.
- Voice-enabled issue input using browser speech recognition (`en-IN`).
- AI booking helper messaging aligned to:
  - Whisper (speech-to-text)
  - GPT-4o-mini (intent parsing)
- Booking form includes payment method option (Razorpay / pay later).

- `login.html`
- `signup.html`
